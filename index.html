Descripción sobre Micro Proyectos Web

Este proyecto está formado por varios proyectos web pequeños; cada uno de estos proyectos es completamente funcional de forma individual. Se puede acceder a los distintos proyecto por los enlaces de esta página, o ingresar al directorio que lo contiene, seleccionando el archivo correspondiente.

La finnalidad de este proyecto es crear documentos de facil acceso, sin necesidad de iniciar un servidor, para comprobar vistas y funciones varias; para el diseño de un sitio web más complejo con un marco de trabajo adecuado que permita crear un proyecto escalable.

En los enlaces a PruebasVarias, se puede acceder a distintas Pruebas de componentes y funciones en documentos html, realizadas durante la elaboración de varios proyectos. Estas pruebas las mantengo en este repositorio para que se puedan utilizar como una plantilla para otros proyectos.
